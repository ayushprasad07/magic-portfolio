## Overview

Wallipop is a wallpaper application for Android that allows users to explore, download, and manage high-quality wallpapers. The app features user authentication, personalized favorites, and a custom upload system to let users contribute their own wallpapers to the community.

## Key Features

- **Wallpaper Browsing & Downloading**: Users can explore a curated collection of wallpapers and download them with a single tap.
- **Favorites Management**: Implemented a local and cloud-synced system for users to save and revisit their favorite wallpapers.
- **Custom Upload System**: Enabled authenticated users to upload their own wallpapers to share with the community.
- **Category-wise Browsing**: Added structured navigation to browse wallpapers by categories and tags.
- **High-Speed Networking**: Used Android Fast Networking library to optimize image loading and API response times.

## Technologies Used

- **Kotlin**: Core programming language for Android app development.
- **Firebase Realtime Database**: For storing wallpaper metadata and user favorites.
- **Firebase Authentication**: To manage secure logins and user-generated uploads.
- **XML**: For designing modern, responsive UI layouts.
- **Android Fast Networking**: To handle efficient network requests and image loading.
- **REST API**: For serving wallpapers dynamically from external or custom APIs.

## Challenges and Learnings

One of the key challenges was optimizing performance while loading high-resolution wallpapers. Implementing efficient image caching and lazy loading improved the user experience significantly. Managing real-time data sync with Firebase for uploads and favorites also provided valuable insights into cloud data handling.

## Outcome

Wallipop offers a visually rich and interactive experience for wallpaper lovers. Its sleek UI, fast performance, and custom upload capability make it a unique addition to the wallpaper app space. It laid the groundwork for future features like wallpaper auto-rotation, dark mode, and community moderation tools.
